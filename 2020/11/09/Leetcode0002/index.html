Leetcode 0002

收获：对于链表的节点，游标的思考。

（1）当需要构造一个链表的时候，可以用一个游标cursor指向根节点。

（2）取到一个新结点的值以后，将该值赋予cursor->next。如下代码，由于构造方法里有这样一个构造器，所以会自动构造一个节点，连接上cursor的next。

ListNode(int val) &#123; 
    this.val = val; 
&#125;

（3）将cursor指向其next。cursor原本引用的是root的地址值，现在让其指向next，就相当于让cursor引用root的next的地址值，也即步骤（2）中新结点的地址值。这样做，可以让我们的链表继续向后添加，并且也不会对原有链表的节点造成破坏。

（4）在步骤（2）里，是不能让cursor直接变成新结点的。否则步骤（3）中，cursor的next是空节点，而cursor又指向其next，这时等于清空了root值，导致链表一直进行“ cursor变为新结点-> cursor指向next -> cursor变为空 -> cursor变为新结点…. ”的死循环。

我的答案：

/**
 * Definition for singly-linked list.
 * public class ListNode &#123;
 *     int val;
 *     ListNode next;
 *     ListNode() &#123;&#125;
 *     ListNode(int val) &#123; this.val = val; &#125;
 *     ListNode(int val, ListNode next) &#123; this.val = val; this.next = next; &#125;
 * &#125;
 */
class Solution &#123;
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) &#123;
        ListNode root = new ListNode(0);
        ListNode cursor = root;
        int more = 0;
        int val = 0;
        while ( l1 != null || l2 != null || more != 0) &#123;
            if (l1 == null  && l2 == null) &#123;
                val = more;
                more = 0;
            &#125;else if (l1 == null) &#123;
                // val是当前位应该存放的数字，more是进一位的值
                val = (l2.val + more) % 10;
                more = (l2.val + more) / 10;
            &#125;else if (l2 == null) &#123;
                // val是当前位应该存放的数字，more是进一位的值
                val = (l1.val + more) % 10;
                more = (l1.val + more) / 10;
            &#125;else&#123;
                // val是当前位应该存放的数字，more是进一位的值
                val = (l1.val + l2.val + more) % 10;
                more = (l1.val + l2.val + more) / 10;
            &#125;

            cursor.next = new ListNode(val);
            cursor = cursor.next;

            if (l1 != null) &#123;
                l1 = l1.next;
            &#125;
            if (l2 != null) &#123;
                l2 = l2.next;
            &#125;

        &#125;
        return root.next;
    &#125;

官方答案：

class Solution &#123;
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) &#123;
        ListNode head = null, tail = null;
        int carry = 0;
        while (l1 != null || l2 != null) &#123;
            int n1 = l1 != null ? l1.val : 0;
            int n2 = l2 != null ? l2.val : 0;
            int sum = n1 + n2 + carry;
            if (head == null) &#123;
                head = tail = new ListNode(sum % 10);
            &#125; else &#123;
                tail.next = new ListNode(sum % 10);
                tail = tail.next;
            &#125;
            carry = sum / 10;
            if (l1 != null) &#123;
                l1 = l1.next;
            &#125;
            if (l2 != null) &#123;
                l2 = l2.next;
            &#125;
        &#125;
        if (carry > 0) &#123;
            tail.next = new ListNode(carry);
        &#125;
        return head;
    &#125;
