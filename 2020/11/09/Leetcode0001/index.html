收获：对于去重，查找唯一的思考。

由于当天没有准时写笔记，导致思路有些模糊。

（1）寻找匹配的两个数，如果用两层for循环，时间复杂度会非常高。而hashmap有个containsKey方法，很高效。并且hashmap还有去重的效果。

我的答案：

class Solution &#123;
    public int[] twoSum(int[] nums, int target) &#123;
        HashMap<Integer, Integer> hashmap = new HashMap<>();

        for(int i=0;i<nums.length;i++)&#123;
            if (hashmap.containsKey(target - nums[i]))&#123;
                return new int []&#123; hashmap.get(target - nums[i]),i&#125;;
            &#125;else&#123;
                hashmap.put(nums[i],i);
            &#125;
        &#125;

    &#125;
